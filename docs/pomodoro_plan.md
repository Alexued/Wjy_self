# 番茄钟功能 - 详细设计与任务规划

## 一、功能清单（对标"番茄Todo"）

### 核心功能
1. **番茄计时器** - 专注25分钟 → 休息5分钟 → 循环，每4个番茄后长休息15分钟
2. **待办任务绑定** - 将番茄钟关联到具体学习任务（可复用计划表数据）
3. **专注统计** - 今日/本周/本月专注时长、番茄数、完成率
4. **专注历史** - 历史记录列表，按日期分组
5. **自定义设置** - 专注/休息/长休息时长、自动开始下一个、提醒方式
6. **白噪音** - 内置多种环境音（雨声、咖啡馆等），专注时播放
7. **悬浮窗计时** - 最小化后以悬浮窗显示倒计时（类似悬浮球风格）
8. **通知提醒** - 番茄完成/休息结束时通知+声音提醒

### 辅助功能
9. **标签分类** - 为番茄添加标签（行测、申论、面试等）
10. **每日目标** - 设置每日番茄目标数，显示进度
11. **成就系统** - 连续打卡天数、累计专注时长里程碑

## 三、技术架构

### 数据库设计 (pomodoro.db)

**focus_sessions 表** - 专注记录
| 字段 | 类型 | 说明 |
|------|------|------|
| id | INTEGER PK | 自增ID |
| task_title | TEXT | 任务名称 |
| tag | TEXT | 标签（行测/申论/面试等） |
| duration_minutes | INTEGER | 实际专注时长(分钟) |
| target_minutes | INTEGER | 目标时长(分钟) |
| is_completed | INTEGER | 是否完成(0/1) |
| plan_task_id | INTEGER | 关联计划表任务ID(可空) |
| started_at | INTEGER | 开始时间(epoch ms) |
| finished_at | INTEGER | 结束时间(epoch ms) |
| created_at | INTEGER | 创建时间 |

**daily_stats 表** - 每日统计（可选，或通过聚合查询）
| 字段 | 类型 | 说明 |
|------|------|------|
| id | INTEGER PK | |
| date | TEXT | YYYY-MM-DD |
| total_focus_minutes | INTEGER | |
| tomato_count | INTEGER | |
| completed_count | INTEGER | |
| target_count | INTEGER | |
