# 陪你刷

公务员考试行测刷题助手。悬浮球截图识别题目，AI 智能解析，错题自动收录。

## 功能

- **悬浮球搜题** - 点击悬浮球截图，自动 OCR 识别并调用 AI 解析
- **错题本** - 自动收录错题，支持题库匹配、导出长图、AI 解析
- **知识卡片** - 闪卡式记忆，支持分类管理和导入导出
- **学习计划** - 日历视图管理学习任务
- **番茄钟** - 专注计时，支持白噪音和应用拦截
- **词典查询** - 成语、词语、歇后语即时查询

## 安装

1. 下载 `陪你刷-v1.0.0-release.apk`
2. 传到手机，打开安装（允许"未知来源"）
3. 首次启动需要授予悬浮窗和录屏权限

## 设置 AI 模型

1. 打开应用，进入 **AI模型** 页面
2. 填写：
   - **API 地址**：`https://api.deepseek.com`（或其他兼容接口）
   - **API Key**：你的密钥
   - **模型**：`deepseek-chat`
3. 点击保存

## 使用方法

### 搜题

1. 主页开启 **悬浮球**
2. 打开任意刷题 APP
3. 点击悬浮球 → 框选题目区域
4. 等待 AI 解析，结果以卡片形式展示

### 错题本

- 搜题时自动匹配题库，匹配成功自动收录
- 点击悬浮球可选择 **记录错题** 模式（不调用 AI，直接收录）
- 错题详情页支持：
  - 查看题干、选项、答案、解析
  - 切换 **题库解析 / AI解析** Tab
  - **导出长图** 分享
  - 编写总结笔记

### 番茄钟

1. 进入 **番茄钟** 页面
2. 选择任务开始专注
3. 可开启 **应用拦截** 屏蔽干扰 APP
4. 支持白噪音（雨声/咖啡厅/森林）

### 知识卡片

- 支持 7 个分类：成语、时政、政治理论、申论、常识、人物素材、面试素材
- 可手动添加或批量导入
- 卡片式浏览，适合碎片时间记忆

## 题型支持

| 题型 | AI 解析 | 视觉模式 |
|------|---------|----------|
| 片段阅读 | ✅ | ❌ |
| 逻辑填空 | ✅ | ❌ |
| 语句表达 | ✅ | ❌ |
| 图形推理 | ✅ | ✅ |
| 定义判断 | ✅ | ❌ |
| 类比推理 | ✅ | ❌ |
| 逻辑判断 | ✅ | ❌ |

> 图形推理使用视觉模式，直接将截图发送给 AI 分析，无需 OCR。

## 技术栈

- Kotlin + Android SDK (minSdk 26)
- PaddleOCR v5 本地识别
- WCDB (SQLite FTS5 全文检索)
- OkHttp 网络请求
